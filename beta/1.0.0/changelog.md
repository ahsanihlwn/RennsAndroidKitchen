# Renn's Android Kitchen

**Renn's Android Kitchen** is a developer tool for modding and repacking Android system images — partition editing, erofs unpacking, and more, all in one place.

---

## v1.0.0 — Beta
*Initial beta release*

### Core
- First Beta release of Renn's Android Kitchen
- Project-based workspace — create and manage multiple image projects
- **erofs**,**EXT2/3/4** unpack and repack support
- Dual file tree mode for side-by-side partition editing

### UI
- Dark interface built with **PySide6**
- Profile system with verified account badges
- Integrated update agent with auto changelog fetch

### Notes
This is an early beta. Expect rough edges. Report bugs to the dev team.
